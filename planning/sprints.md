# Sprint Plan

## Sprint 1

- Story: As a new user i want to be able to create an account
- Acceptance criteria: Given valid account details, when i create an account, an account is created and i am logged in within 2 seconds
- Test case:
        Precondition: User does not exist within the database
        Input: Valid email and password for account creation
        Action: Submit account creation form
        Expected result: Account created and user logged in within 2 seconds
  
- Story: As a user i want to be able to input height, weight, and activity level
- Acceptence criteria: Given acceptable measurements, when information is inputted, the data is saved to the database within 3 seconds
- Test case:
        Precondition: There are upper and lower limits of what data can be inputted
        Input: Valid height, weight, and activity level
        Action: Submit Data
        Expected result: Data is saved to the database within 3 seconds
- Story: As a user i would like to easily be able to navigate through the system
- Acceptence criteria: Given a new user, i should be able to navigate through the system easily without help
- Test case:
        Precondition: It should be possible to navigate through the site
        Input: Mouse movement and switching tabs
        Action: Clicking on tabs
        Expected result: Tab is changed instantly for the user
